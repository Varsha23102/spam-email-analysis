# 📧 Spam Email Analysis

## 📌 Overview

A data analysis project to identify spam emails using exploratory data analysis (EDA) and a basic machine learning model.

---

## 🛠️ Tech Stack

Python, Pandas, Matplotlib, Scikit-learn, WordCloud

---

## 📊 Key Highlights

* Performed data cleaning and preprocessing on email text
* Analyzed spam vs non-spam distribution
* Created visualizations (bar chart, histogram, wordcloud)
* Built a Naive Bayes classifier for spam detection

---

## 📈 Results

* Achieved ~98% accuracy
* Identified common spam patterns and keywords

---

## 🧪 Sample Output

Input: You won a free lottery prize
Output: 🚫 Spam Email
